## Overview

This repository contains artwork for the [OQTOPUS](https://oqtopus-team.github.io/) project.
For details on use of the artwork, please see the [style guide](OQTOPUS_GUIDELINES.pdf).

- AI: [Adobe Illustrator files](./AI/)
- PNG: [PNG files](./PNG/)
- SVG: [SVG files](./SVG/)
